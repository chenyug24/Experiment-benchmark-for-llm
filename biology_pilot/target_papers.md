## Review Checklist

For each target paper, verify:

- earliest public availability date
- DOI / PMID / PMCID
- whether there was an earlier preprint
- whether each gold label is explicitly supported by results, figures, or tables
- whether each negative control is truly unsupported or context-shifted
- whether prior-literature references are available before the cutoff
